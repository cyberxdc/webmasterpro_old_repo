== Description ==
WebMasterPro is a comprehensive WordPress plugin designed to enhance your website's security and customization options. It offers various features including server info, SMTP settings, shortcode management, security enhancements, customization options, activity logs, Contact Form 7 database management, and update/license management.

== Features ==
1. **Dashboard**
   - Server Info
   - SMTP Settings and Send Test Email

2. **Shortcodes Submenu**
   - Login, Register, Forgot Password Shortcodes
   - Custom Shortcodes Display

3. **Security Submenu**
   - General Security
   - Files Security
   - User Security
   - Database Security
   - Firewalls
   - Two-Factor Authentication (2FA)

4. **Customization**
   - Login Page Customization
   - Custom Styles and Scripts

5. **Settings**
   - General Settings
   - File Upload Settings
   - Media Settings
   - Page Settings
   - Post Settings
   - Help and Support

6. **Logs and Activity**
   - User Logs
   - Visitors Logs

7. **Contact Form 7 Database**

8. **Update and License**

== Installation ==
1. Upload the `cyberxdc` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
